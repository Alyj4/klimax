# KLIMAX

Sound built to cut. A minimal black-and-red site for showcasing original tracks.

## Structure

- `index.html` — page markup
- `assets/style.css` — styling (black/red, sharp angular aesthetic)
- `assets/script.js` — custom audio player logic
- `assets/audio/` — track files

## Running locally

Open `index.html` in a browser, or serve the folder:

```
python3 -m http.server 8000
```

## Deploying

Pushed to GitHub Pages from the `main` branch, `/ (root)`.
